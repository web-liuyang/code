# 各种代码集合列表

代码是网络上收集的方便大家使用

1. **brank** 各类银行代码
2. **city** 各类城市代码
3. **country** 各类国家代码
4. **energy** 汽车源动力代码
5. **goods** 物流运输物品分类代码
6. **insurance** 各类保险公司的代码
7. **license-plate** 汽车牌照颜色代码
8. **pay** 支付方式代码
9. **route** 配送运输方式代码
10. **transport-organization** 各类物流组织代码
11. **vehicleType**. 各种货车类型代码
